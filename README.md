# bank-cli

Simulates a banking system. Runs on CLI
Features:
1. Login
2. Create Account
3. Deposit/Witdrawal funds
4. Check Balance
5. Transaction History
6. JSON file Storage

## Setup

Must have NodeJS and NPM installed on your computer
```sh
$ npm install
$ node index.js
```
